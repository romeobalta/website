## How to run

Clone the repo:

```bash
git clone git@github.com:romeobalta/website.dev.git
```

Install packages:

```bash
npm install
# or
yarn
# or
bun install
```

Run in dev mode:

```bash
npm run dev
# or
yarn dev
# or
bun dev
```
